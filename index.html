Final Projects

Overview


Project Goal: Work on a team to design a website of your choosing that showcases your prowess with HTML, CSS, and JavaScript and your ability to learn and leverage new, advanced Web Development concepts to make a unique website.


Course Description: This beginning course introduces the basic ideas of computing via the World Wide Web through the creation of dynamic web pages. Three layers are built: HTML, for document structure, CSS for document appearance, and JavaScript for page behavior. JavaScript, a full-featured, Turing-complete programming language, is used to learn the fundamental components of programming: variables, objects, functions, conditional logic, and iteration. In-class individual and group work culminates in an individual or group project chosen by the students.


Learning Outcomes:

  • Describe how a web page is transmitted and explain the relationship between web pages, web servers, and databases.
  • Use correct HTML to create a web page that displays in a browser and validates.
  • Manipulate a webpage layout and appearance using CSS.
  • Manipulate the behavior of a webpage with JavaScript using variables, objects, functions, conditional logic, and iteration.
  • Establish communication among page elements.
  • Create an interactive website using HTML, CSS, and JavaScript.
